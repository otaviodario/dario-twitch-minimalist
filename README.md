# Dario Twitch Minimalist

A lightweight Firefox extension designed to improve performance on low-end hardware by redirecting Twitch channels to[CoolTwitch](https://api.roaringiron.com/cooltwitch/).

## Features
- **Manual Redirection:** Quickly switch to a minimal viewer via the extension popup.
- **Auto-Redirect:** Persistent option to automatically load Twitch channels in a lightweight mode.
- **Privacy First:** No data collection, no telemetry, and no tracking.
- **Bilingual:** Fully supports English and Portuguese.

## How to install
*Currently under review by Mozilla. Once approved, you will be able to install it directly from the Firefox Add-ons store.*
