# Dario Twitch Minimalist
A lightweight Firefox extension that redirects Twitch channels to CoolTwitch for better performance on low-end hardware.

## Features
- Manual redirection to CoolTwitch via popup.
- Auto-redirect feature for a smoother browsing experience.
- Lightweight & Privacy-focused: No tracking or data collection.
- Bilingual Support (EN/PT-BR).
